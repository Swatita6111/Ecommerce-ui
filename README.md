# Product Explorer Dashboard

A modern e-commerce dashboard built with Next.js, TypeScript, and Tailwind CSS.  
Explore products, filter by category, mark favorites, and view product details in a responsive UI.

---

## Tech Stack
- **Frontend:** Next.js (App Router), TypeScript
- **Styling:** Tailwind CSS
- **State Management:** React `useState` & localStorage
- **Other Features:** Responsive UI, Dark/Light mode support

---

## Features
- Product listing with search and category filter
- Product details page with image, description, and price
- Favorite products stored in `localStorage`
- Responsive design for mobile and desktop
- Loading and error states handled gracefully
- **Dark/Light mode toggle**

### Not Required, But Implemented / Strong Signals
- Server Components used where appropriate
- Pagination for product listings
- Sorting products by price
- Basic accessibility improvements (ARIA labels, keyboard navigation)
- Unit tests using Jest and React Testing Library
